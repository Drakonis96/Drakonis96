# 📚 Featured Projects

A selection of my main repositories and ongoing work:

- [**ocrAI**](https://github.com/Drakonis96/ocrai)  
  Combines OCR and AI to process, enhance, and translate documents. Recognized text is embedded directly into the original PDF, with optional translation output.  
  _Python · JavaScript_

- [**dubyDB**](https://github.com/Drakonis96/dubyDB)  
  A self-hosted workspace for flexible databases, file management, analytics, backups, and API integrations. Inspired by tools like Notion, but fully under your control.  
  _JavaScript · CSS_

- [**NotyVisualScan**](https://github.com/Drakonis96/notyvisualscan)  
  Uses AI to generate descriptions and tags for images in Notion, including automated upload and organization workflows.  
  _Python · HTML_

- [**WhisPad**](https://github.com/Drakonis96/whispad)  
  Converts voice into structured notes using local or cloud-based Whisper models, with AI-powered text enhancement and formatting.  
  _C++_

- [**Statainer** (formerly DockerStats)](https://github.com/Drakonis96/statainer)  
  A focused web dashboard for monitoring and managing Docker containers. Includes real-time logs, notifications, advanced filtering, and update management.  
  _HTML · Python_

- [**PlexyTrack**](https://github.com/Drakonis96/plexytrack)  
  Syncs history, lists, and ratings between Plex, Trakt, and Simkl. Also supports backup and restore of Trakt data.  
  _Python · HTML · CSS_
